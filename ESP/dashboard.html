📋 Setup Steps

  1. Flash master/master.ino to Master ESP32. Open Serial Monitor at 115200 baud.
  2. Note the MAC address printed on boot (e.g. A4:CF:12:98:EF:01).
  3. Paste that MAC into masterMAC[] inside slave1/slave1.ino, then flash it to the Slave ESP32.
  4. Close Serial Monitor, then open this file via VS Code Live Server in Chrome/Edge.
  5. Click Connect Master (Serial), select the Master COM port, and watch the tank fill live!
